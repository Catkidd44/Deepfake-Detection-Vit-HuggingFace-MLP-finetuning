# Deepfake-Detection-Vit-HuggingFace-MLP-finetuning
Fine-tuned a Hugging Face ViT model with a custom MLP head on a Kaggle video dataset. Extracted 30 frames per video to identify spatial anomalies, achieving high accuracy through targeted deep learning. This was a group project.
Data from: https://www.kaggle.com/datasets/reubensuju/celeb-df-v2
Medium Blog : https://medium.com/@yunyun.zhu/seeing-through-celebrity-deepfakes-408da821aafe
